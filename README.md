## üõ†Ô∏è Installation

1.  **Prerequisites:**
    * Python 3.8 or higher.

2.  **Install Dependencies:**
    Open your terminal/command prompt and run:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Asset Setup:**
    For the full visual experience, ensure the following files are in the script directory:
    * `spectre.png` (Background image)
    * `spectre.ico` (Window icon)

## üöÄ Usage

1.  Run the application:
    ```bash
    python dist/spectre5.py
    ```
2.  **Select Target:** Click **Browse** to select a file or folder to scan.
3.  **Scan:** Click **MINE SECRETS** to begin the thread-based scan.
4.  **Review:** Results will appear in the scrollable area, sorted by severity.
